# CustomSideMenu
サイドメニュー表示して、アニメーションを実装するプロジェクト
## このリポジトリの目的
.matchedGeometryEffectやサイドメニューをアニメーション化する方法を学習する

1つの画面をViewファイルを分割して作成する
## このサンプルで学んだこと
 .matchedGeometryEffectを使用することで、Heroアニメーション(シームレスに移動したように見せるアニメーション)を実装することが出来る
